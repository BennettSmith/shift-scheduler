# Shift Scheduler (Troop 900 Tree Lot)

Mobile-first scheduling app for managing volunteer shifts for Troop 900's Christmas tree lot fundraiser.

## Tech Stack

- **iOS**: Swift, SwiftUI, Clean Architecture with Swift Packages, Firebase iOS SDK
- **Backend**: Firebase (Firestore, Auth, Functions, Cloud Messaging)
- **CI/CD**: GitHub Actions (to be added)

## Project Structure (Planned)

- `ios/` – iOS application (Xcode project, Swift packages)
- `functions/` – Firebase Cloud Functions (TypeScript, Clean Architecture)
- `android/` – Android application (empty for now, to be added later)
- `planning/` – Architecture and setup documents
